# Enterprise Security Baseline

Enterprise Security Baseline is an open-source initiative that provides practical, auditable security standards for cloud infrastructure, Linux systems, and application delivery pipelines.

The project focuses on translating established security frameworks into actionable controls that can be applied consistently across enterprise environments.

## Who This Is For
- Security Architects
- Cloud and Platform Engineers
- DevSecOps Engineers
- Risk and Compliance Teams

## Scope
This repository covers:
- Security governance and risk ownership
- Threat modelling and secure design principles
- Cloud security baselines (AWS, Azure)
- Linux system hardening and auditing
- Secure CI/CD and application security practices

## Design Philosophy
Security controls should be practical, measurable, and repeatable.
This project prioritises real-world implementation over theoretical guidance.

## Status
Initial release. Actively developed and open to peer review and contributions.
